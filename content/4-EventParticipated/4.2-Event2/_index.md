### Event Objectives

- Introduce the FLOCI platform and the emerging trend of AI Agents in software development.
- Share how AI Workflows can improve software development efficiency.
- Explain how AI can be integrated into Cloud Computing and DevOps workflows.
- Connect participants interested in AI, Cloud Computing, and Software Engineering.

### Main Topics

#### Introduction to FLOCI

- FLOCI is an open-source platform for building and managing AI Workflows.
- It enables multiple AI Agents to collaborate within a single workflow.
- The platform supports flexible and scalable AI application development.

#### AI Workflow

- The speaker demonstrated how a complex problem can be divided into smaller processing steps.
- Different AI Agents can perform different responsibilities such as requirement analysis, code generation, and result validation.
- This workflow improves automation and development efficiency.

#### Applications in Software Development

- AI can assist in project planning, documentation, and software development.
- Workflow-based AI provides better organization than relying on a single AI model.
- Developers remain responsible for validating and reviewing AI-generated outputs.
